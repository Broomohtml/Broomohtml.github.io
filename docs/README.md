# Finance App

Budget planner mensile personale — PWA installabile su iPhone.

**App live:** https://broomohtml.github.io

---

## Avvio locale

1. Apri la cartella in VS Code
2. Click destro su `index.html` → **Open with Live Server**
3. `F12` → icona 📱 per la vista mobile

## Installazione su iPhone

1. Apri **Safari** → `https://broomohtml.github.io`
2. **Condividi** → **Aggiungi a schermata Home**
